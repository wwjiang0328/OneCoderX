使用说明

三分钟上手：配置目录、开始整理、统一管理与编辑笔记

1

初始化设置

在“设置”页配置图片、视频、音频、文档、Markdown、其他目录。

2

拖拽文件整理

把文件拖到主界面（Windows 可用加速球），系统自动识别并归类到目标目录。

3

剪切板图片处理

对无法直接拖拽的截图，点击“识别并整理剪切板图片”即可快速落盘归档。

4

资源管理筛选

在“资源管理”中按分类、标签、时间、大小进行筛选和排序，快速定位文件。

5

Markdown 笔记编辑

在“Markdown 编辑”中管理文档树、编辑内容并结合大纲导航提升写作效率。

6

检查更新

在“设置”页可检测新版本并一键跳转下载页面，保持应用持续更新。

技术架构

稳定的桌面底座 + 现代前端 + 本地数据库，保障性能与可维护性

🖥️

Electron

负责主进程能力与系统集成：文件移动、托盘、窗口管理、IPC 通信。

Vue 3

组合式 API 构建多视图前端，配合 Vue Router 组织“整理 / 资源 / 笔记”模块。

⚡

Vite

提供极速开发体验与高效构建，协同 vite-plugin-electron 管理桌面端构建流程。

📦

Pinia

统一管理配置、日志与前端状态，提升模块协作与维护效率。

🗄️

SQLite

本地资源索引与标签关系存储，支撑筛选、排序、置顶、星标与日志查询。

📝

Vditor + chokidar

Vditor 提供 Markdown 编辑能力，chokidar 负责文件变更监听与索引同步。
