发展历程

1991–1995：起源与诞生

James Gosling 等人在 Sun 启动 Green 项目，语言初名 Oak，后改为 Java。1995 年正式发布 1.0，承诺“Write Once, Run Anywhere”。

1996–2006：企业级崛起与经典版本

引入 Collections、Swing、JDBC、RMI 等。Java 5（2004）带来泛型、枚举、注解、自动装箱；Java 6 强化性能与脚本支持。

2008–2014：模块化与函数式革命

Java 8（2014）引入 Lambda、Stream API、Optional、新的 Date/Time API，成为现代 Java 分水岭。

2017–2021：模块系统与长期支持

Java 9 引入模块系统（Project Jigsaw）；Java 11 成为首个长期支持版（LTS），移除旧模块。

2021–2026：并发与性能新时代

Java 17/21 LTS 引入密封类、记录、模式匹配、虚拟线程（Project Loom）；2025–2026 年聚焦值类型（Valhalla）、外部函数与内存 API（Panama）、结构化并发等。
